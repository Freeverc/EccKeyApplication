*****************************************************************************************************
ECCKeyGenerator.exe 用于生成数据库
ECCKeyRetriver.exe 用于从数据库检索公钥
BigNumCalculator.exe 用于大数四则运算
ECCKeyDoubler.exe 用于从数据库倍增检索
ECCPointCalculator 用于椭圆曲线有限素数域上点的加法和数乘
*****************************************************************************************************

*****************************************************************************************************
ECCKeyGenerator：
默认的G为:(比特币标准secp256k1)
X coordinate = 79BE667EF9DCBBAC55A06295CE870B07029BFCDB2DCE28D959F2815B16F81798
Y coordinate = 483ADA7726A3C4655DA4FBFC0E1108A8FD17B448A68554199C47D08FFB10D4B8
用户需要输入:
初始点 G 的 x 坐标
初始点 k 值
结束点 k 值
数据库以字节块形式存储，名称为"publicKeys.kdb", 可视化数据为"publicKeysVisual.txt"
Ctrl - C 终止程序
控制台输出花费时间
本文件夹下的数据库构建于：
start of k = 1FFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF00000
 end  of k = 20000000000000000000000000000000000000000000000000000000000FFFFF

*****************************************************************************************************
ECCKeyDoubler
输入初始点G的x坐标，在数据库搜索，如果找到则输出搜索次数n, 当前私钥key(2^n)，以及对应的x和y
否则更新G = 2 * G， n = n+1

默认的G为:(比特币标准secp256k1)
X coordinate = 79BE667EF9DCBBAC55A06295CE870B07029BFCDB2DCE28D959F2815B16F81798
Y coordinate = 483ADA7726A3C4655DA4FBFC0E1108A8FD17B448A68554199C47D08FFB10D4B8
用户需要输入:
初始点 G 的 x 坐标
初始点 k 值
数据库以字节块形式存储，名称为"retriveKeys.pdb", 可视化数据为"retriveKeysVisual.txt"
按空格键可以暂停或者继续，Ctrl - C 终止程序
搜索结束后控制台输出花费时间
*****************************************************************************************************

*****************************************************************************************************
ECCPointCalculator
支持secp256k1椭圆曲线上定义的加法和数乘(+, *), 输入符号和坐标，采用数据均为十六进制
