*******************************************************************************
ECCKeyGenerator.exe 用于生成数据库
ECCKeyRetriver.exe 用于从数据库检索公钥
*******************************************************************************

*******************************************************************************
ECCKeyGenerator：
默认的G为:(比特币标准secp256k1)
X coordinate = 79BE667EF9DCBBAC55A06295CE870B07029BFCDB2DCE28D959F2815B16F81798
Y coordinate = 483ADA7726A3C4655DA4FBFC0E1108A8FD17B448A68554199C47D08FFB10D4B8
用户需要输入:
初始点 G 的 x 坐标
初始点 k 值
结束点 k 值
数据库以字节块形式存储，名称为"publicKeys.txt"
Ctrl - C 终止程序
控制台输出花费时间
本文件夹下的数据库构建于：
start of k = 20C05E60575929A4F7E7F6E3B0E31124B99FE1492BE497726E0D3D7B33D00000
 end  of k = 20C05E60575929A4F7E7F6E3B0E31124B99FE1492BE497726E0D3D7B33DFFFFF

*******************************************************************************
ECCKeyRetriver
默认的G为:(比特币标准secp256k1)
X coordinate = 79BE667EF9DCBBAC55A06295CE870B07029BFCDB2DCE28D959F2815B16F81798
Y coordinate = 483ADA7726A3C4655DA4FBFC0E1108A8FD17B448A68554199C47D08FFB10D4B8
用户需要输入:
初始点 G 的 x 坐标
初始点 k 值
数据库以字节块形式存储，名称为"retriveKeys.txt"
按空格键可以暂停或者继续，Ctrl - C 终止程序

本文件夹下的检索结果数据库构建于：
